# Puzzle Canvas

无限画布，粘贴网址、图片、视频、文本即可收藏。

## 功能

- **粘贴收藏** — 粘贴 URL 自动抓取标题、描述、封面图；粘贴图片/视频直接上传；粘贴文本创建文本节点
- **拖拽上传** — 拖拽图片或视频文件到画布，自动以瀑布流排列
- **无限画布** — 自由缩放、平移，视口状态自动保存
- **多选操作** — 按住 Space + 拖拽框选节点，支持批量整理、下载、分享、删除
- **节点分组** — 将多个节点设为小组，虚线框标注，拖拽小组同步移动组内所有节点
- **节点分享** — 选中节点生成分享密钥，其他用户输入密钥或链接即可导入（瀑布流排布）
- **持久化** — 所有节点和位置自动保存到 SQLite 数据库

## 技术栈

| 层 | 技术 |
|---|---|
| 前端 | React 19 · @xyflow/react · Lucide Icons · Vite |
| 后端 | Hono · Bun |
| 数据库 | SQLite · Drizzle ORM |

## 快速开始

```bash
# 安装依赖
bun install

# 初始化数据库
bun run db:push

# 启动开发（前端 + 后端）
bun run dev:all
```

- 前端：http://localhost:5175
- 后端：http://localhost:4001

## 操作说明

| 操作 | 说明 |
|---|---|
| `Ctrl/⌘ + V` | 粘贴 URL、图片、视频或文本 |
| 拖拽文件到画布 | 上传图片/视频 |
| `Space + 拖拽` | 框选多个节点 |
| `Shift + 点击` | 切换单个节点的选中状态 |
| `Delete` | 删除选中的节点 |
| 滚轮 | 缩放画布 |

### 分享操作

| 操作 | 说明 |
|---|---|
| 选中节点 → 点击「分享」 | 生成 8 位分享密钥和链接 |
| 复制密钥/链接 | 发送给同事 |
| 右侧工具栏 → 点击「导入」 | 打开导入弹窗 |
| 输入密钥或链接 → 点击「导入」 | 节点以瀑布流排布导入到画布 |

### 小组操作

| 操作 | 说明 |
|---|---|
| 选中 2+ 节点 → 点击「设置小组」 | 将选中节点创建为小组 |
| 点击小组空白区域 | 聚焦小组，显示操作栏（整理、重命名、取消分组） |
| 拖拽小组 | 同步移动组内所有节点 |
| 拖拽节点出小组 | 自动从小组中脱离 |
| 点击「整理」 | 将组内节点瀑布流排列并自动调整小组边界 |
| 点击画布空白处 | 取消小组聚焦 |
